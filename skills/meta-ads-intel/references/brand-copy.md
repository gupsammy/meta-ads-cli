# Brand Copy Reference

Framework for evaluating and generating Meta ad copy. Used during creative analysis (Step 7). User-specific brand context (product, audience, hooks) is in `~/.meta-ads-intel/brand-context.md`.

## The Four Horsemen (Kallaway Framework)

Every ad pulls at least one psychological lever. Winning copy rotates across these so Meta tests psychological angles, not just wording variations.

| Horseman | Core Trigger | Opening Pattern |
|----------|-------------|-----------------|
| **Money** | Revenue loss, ROI, cost savings | "Every [thing] you miss is worth $X." |
| **Time** | Wasted hours, efficiency, speed | "Your [person] isn't a [wrong role]." |
| **Status** | Peer comparison, industry moves | "X+ [peers] already [did thing]." |
| **Fear** | Loss aversion, competitive threat | "The [competitor] down the street already has this." |

## Image Format to Psychology Mapping

Match copy's psychological lever to what the image communicates. Copy reinforces the image — never repeats it.

| Image Format | Lead With | Copy Tone |
|-------------|-----------|-----------|
| Notes app / screenshot | Fear + Money | Casual first-person discovery |
| Receipt / POS | Money + Time | Specific, transactional |
| Before/after split | Money + Time | Contrast, transformation |
| Review / testimonial | Status + Money | Third-person proof |
| Product / lifestyle | Fear + Time | Emotional, atmospheric |
| Data visualization | Money + Status | Authority, numbers-forward |
| Bold text / graphic | Any | Direct, declarative |
| UGC / found footage | Fear + Money | Organic, discovery-style |

## Copy Specs

| Element | Spec | Reason |
|---------|------|--------|
| Headline | 25-40 chars, hard truncation at 50 | Mobile truncation kills CTR |
| Body | 50-120 words | Short enough to read, long enough to convince |
| Description | Max 30 chars | Cut off in most placements |
| Paragraphs | 2-3 blocks, 1-2 sentences each | Walls of text = scroll past |
| Numbers | At least 1 per variant | Specificity = credibility |
| Opening line | Pain, stat, or bold claim | Never brand name first |
| Closing line | Social proof + soft CTA | "X+ already made the switch." |

## Variant Rules

When generating multiple variants:
- Each variant MUST hit a different Horseman
- No two headlines use the same structure
- At least one variant should feel organic (not obviously an ad)
- Cross-reference against account's current top performers

## Forbidden Words

revolutionary, game-changer, cutting-edge, innovative solution, transform your business, unlock your potential, unleash, harness the power, leverage (as verb), next-level, best-in-class, world-class, seamless, robust, synergy
